## Usage with Claude Code

Add to your `~/.claude/settings.json`:

```json
{
  "mcpServers": {
    "mcp-legal-analyzer": {
      "command": "python",
      "args": ["-m", "src.server"],
      "cwd": "<path-to-mcp-legal-analyzer>"
    }
  }
}
```
