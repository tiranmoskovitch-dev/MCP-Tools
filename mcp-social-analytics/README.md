## Usage with Claude Code

Add to your `~/.claude/settings.json`:

```json
{
  "mcpServers": {
    "mcp-social-analytics": {
      "command": "python",
      "args": ["-m", "src.server"],
      "cwd": "<path-to-mcp-social-analytics>"
    }
  }
}
```
