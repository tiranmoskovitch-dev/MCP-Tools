## Usage with Claude Code

Add to your `~/.claude/settings.json`:

```json
{
  "mcpServers": {
    "mcp-color-tools": {
      "command": "python",
      "args": ["-m", "src.server"],
      "cwd": "<path-to-mcp-color-tools>"
    }
  }
}
```
