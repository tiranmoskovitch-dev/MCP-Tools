## Usage with Claude Code

Add to your `~/.claude/settings.json`:

```json
{
  "mcpServers": {
    "mcp-real-estate": {
      "command": "python",
      "args": ["-m", "src.server"],
      "cwd": "<path-to-mcp-real-estate>"
    }
  }
}
```
