# Whop Tiered Pricing — MCP Tools
> Updated: 2026-03-17

## Pricing Strategy

Each MCP tool gets 3 tiers. All tiers include source code and pip-installable package.

---

## Tier Structure (Per Tool)

### Starter — $29
- Full source code
- pip-installable package
- README with setup instructions
- Works with Claude Code, Cursor, Windsurf, any MCP client
- Community support (GitHub Issues)
- Lifetime updates for this tier

### Professional — $79
- Everything in Starter, plus:
- Priority email support (48hr response)
- Docker deployment config (Dockerfile + docker-compose.yml)
- CI/CD pipeline templates (GitHub Actions)
- Advanced configuration examples
- Rate limiting and caching middleware
- Production deployment guide

### Enterprise — $199
- Everything in Professional, plus:
- 1-on-1 setup call (30 min, scheduled via Calendly)
- Custom integration support (connect to your specific APIs/databases)
- Multi-tool orchestration guide (how to chain MCP tools together)
- Priority feature requests
- License for unlimited internal deployments
- Quarterly updates briefing
