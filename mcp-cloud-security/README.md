## Usage with Claude Code

Add to your `~/.claude/settings.json`:

```json
{
  "mcpServers": {
    "mcp-cloud-security": {
      "command": "python",
      "args": ["-m", "src.server"],
      "cwd": "<path-to-mcp-cloud-security>"
    }
  }
}
```
