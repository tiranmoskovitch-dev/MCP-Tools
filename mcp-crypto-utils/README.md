## Usage with Claude Code

Add to your `~/.claude/settings.json`:

```json
{
  "mcpServers": {
    "mcp-crypto-utils": {
      "command": "python",
      "args": ["-m", "src.server"],
      "cwd": "<path-to-mcp-crypto-utils>"
    }
  }
}
```
