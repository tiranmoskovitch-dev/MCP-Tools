## Usage with Claude Code

Add to your `~/.claude/settings.json`:

```json
{
  "mcpServers": {
    "mcp-cicd-optimizer": {
      "command": "python",
      "args": ["-m", "src.server"],
      "cwd": "<path-to-mcp-cicd-optimizer>"
    }
  }
}
```
