## Usage with Claude Code

Add to your `~/.claude/settings.json`:

```json
{
  "mcpServers": {
    "mcp-ecom-analytics": {
      "command": "python",
      "args": ["-m", "src.server"],
      "cwd": "<path-to-mcp-ecom-analytics>"
    }
  }
}
```
