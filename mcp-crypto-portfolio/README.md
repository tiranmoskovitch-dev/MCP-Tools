
## Usage with Claude Code

Add to your `~/.claude/settings.json`:

```json
{
  "mcpServers": {
    "mcp-crypto-portfolio": {
      "command": "python",
      "args": ["-m", "src.server"],
      "cwd": "<path-to-mcp-crypto-portfolio>"
    }
  }
}
```
