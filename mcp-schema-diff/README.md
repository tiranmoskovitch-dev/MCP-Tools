## Installation

```bash
pip install -e .
```

## Usage with Claude Code

Add to your `~/.claude/settings.json`:

```json
{
  "mcpServers": {
    "mcp-schema-diff": {
      "command": "python",
      "args": ["-m", "src.server"],
      "cwd": "<path-to-mcp-schema-diff>"
    }
  }
}
```
