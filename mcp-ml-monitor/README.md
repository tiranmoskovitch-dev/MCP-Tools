## Usage with Claude Code

Add to your `~/.claude/settings.json`:

```json
{
  "mcpServers": {
    "mcp-ml-monitor": {
      "command": "python",
      "args": ["-m", "src.server"],
      "cwd": "<path-to-mcp-ml-monitor>"
    }
  }
}
```
