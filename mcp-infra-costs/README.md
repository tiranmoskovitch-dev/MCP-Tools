
## Usage with Claude Code

Add to your `~/.claude/settings.json`:

```json
{
  "mcpServers": {
    "mcp-infra-costs": {
      "command": "python",
      "args": ["-m", "src.server"],
      "cwd": "<path-to-mcp-infra-costs>"
    }
  }
}
```
