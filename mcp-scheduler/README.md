## Installation

```bash
pip install -e .
```

## Usage with Claude Code

Add to your `~/.claude/settings.json`:

```json
{
  "mcpServers": {
    "mcp-scheduler": {
      "command": "python",
      "args": ["-m", "src.server"],
      "cwd": "<path-to-mcp-scheduler>"
    }
  }
}
```
