## Installation

```bash

```

## Usage with Claude Code

Add to your `~/.claude/settings.json`:

```json
{
  "mcpServers": {
    "mcp-log-analyzer": {
      "command": "python",
      "args": ["-m", "src.server"],
      "cwd": "<path-to-mcp-log-analyzer>"
    }
  }
}
```
