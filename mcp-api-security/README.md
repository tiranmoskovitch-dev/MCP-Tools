## Usage with Claude Code

Add to your `~/.claude/settings.json`:

```json
{
  "mcpServers": {
    "mcp-api-security": {
      "command": "python",
      "args": ["-m", "src.server"],
      "cwd": "<path-to-mcp-api-security>"
    }
  }
}
```
